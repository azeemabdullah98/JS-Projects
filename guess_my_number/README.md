# Guess my number

This game stores any one random number (secret number) between 1 and 20 and ask the user to guess the correct number. 
Score will be awarded based on the number of attempts that the user took to guess the correct number (Max. retry allowed is 20). Highscore will be calculated based on the minimum number of attempts the user took to arrive at the correct secret number.

Using HTML, CSS as Frontend and JavaScript as Backend
